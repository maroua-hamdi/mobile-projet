## 2. Table `cache` (lignes 30-36)

```sql
DROP TABLE IF EXISTS `cache`;
CREATE TABLE IF NOT EXISTS `cache` (
  `key` varchar(191) COLLATE utf8mb4_unicode_ci NOT NULL,
  `value` mediumtext COLLATE utf8mb4_unicode_ci NOT NULL,
  `expiration` int NOT NULL,
  PRIMARY KEY (`key`)
) ENGINE=MyISAM DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

**Explication :**
- **`DROP TABLE IF EXISTS`** : Supprime la table si elle existe déjà (évite les erreurs)
- **`CREATE TABLE IF NOT EXISTS`** : Crée la table seulement si elle n'existe pas
- **Colonnes :**
  - `key` : Clé unique pour identifier l'entrée de cache
  - `value` : Données mises en cache (texte de taille moyenne)
  - `expiration` : Timestamp Unix indiquant quand le cache expire
- **Clé primaire** : Sur `key` pour garantir l'unicité
- **Moteur MyISAM** : Moteur de stockage MySQL rapide pour la lecture

## 6. Table `jobs` (lignes 97-108)

```sql

  `id` bigint UNSIGNED NOT NULL AUTO_INCREMENT,
  `queue` varchar(191) COLLATE utf8mb4_unicode_ci NOT NULL,
  `payload` longtext COLLATE utf8mb4_unicode_ci NOT NULL,

  `reserved_at` int UNSIGNED DEFAULT NULL,

  PRIMARY KEY (`id`),
  KEY `jobs_queue_index` (`queue`)
) ENGINE=MyISAM DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

**Explication :**
- **Fonction** : File d'attente des tâches asynchrones (Laravel Queue)
- **Colonnes :**
  - `id` : Identifiant unique
  - `queue` : Nom de la file d'attente
  - `payload` : Données du job (sérialisées)
  - `attempts` : Nombre de tentatives d'exécution
  - `reserved_at` : Timestamp Unix quand le job a été réservé (NULL si disponible)
  - `available_at` : Timestamp Unix quand le job devient disponible
  - `created_at` : Timestamp Unix de création
- **Index** : Sur `queue` pour accélérer les recherches par file d'attente
